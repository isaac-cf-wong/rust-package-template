# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.
Participants are expected to act respectfully and constructively in project
spaces.

Report unacceptable behavior to the project maintainers through the repository
contact channels. Reports will be reviewed promptly and fairly.

The full Contributor Covenant text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>.
